# KSJ2 Address Salvage Tool (KSJ2:ADS サルベージツール)

OpenStreetMap (OSM) の過去履歴 (v1) から `KSJ2:ADS` タグを取得し、現在住所情報 (`addr:*`) を持たない KSJ2 由来の POI (Node) に対して `addr:full` を付与するための `.osc` (osmChange) ファイルを生成するツールです。

## 目的

日本の国土数値情報 (KSJ2) 由来のデータにおいて、過去には存在していたが現在のバージョンでは失われてしまった住所情報を復元し、OSMデータの品質を向上させることを目的としています。

## 主な機能

- **対象抽出**: Overpass API を使用して、指定された都道府県内で `source=KSJ2` かつ `amenity` タグを持ち、`addr:*` の住所情報を持たない Node を自動抽出します。
- **履歴解析**: OSM History API を使用して、各ノードの Version 1 の情報を取得し、`KSJ2:ADS` (住所) や `KSJ2:PubFacAdmin` (管理者) タグをサルベージします。
- **行政界特定**: ノードの現在位置から Overpass API を使用して現在の市区町村名を特定し、正確な `addr:full` を構築します。
- **位置変化検証**: オリジナル (v1) と現在の位置を比較し、位置が大きく変化している場合の不適切なタグ付与を防止します。
- **osmChange 出力**: JOSM等で読み込み・アップロード可能な `.osc` 形式で変更点を出力します。

## 動作要件

- Java 21 以上
- Maven (ビルド用)
- インターネット接続 (OSM/Overpass API へのアクセス)

## ビルド方法

プロジェクトのルートディレクトリで以下のコマンドを実行してください。

```bash
mvn clean package
```

`target/ksj2tool-0.x.x.jar` (実行可能JAR) が生成されます。

## 使い方

JARファイルを実行し、引数に対象の都道府県名を指定します。

```bash
java -jar target/ksj2tool-0.x.x.jar 東京都
```

※ 引数は「東京都」「神奈川県」「山梨県」などのJIS表記で指定してください。
※ 「全国」を指定すると、全国を対象に処理を試みますが、APIのタイムアウトや負荷に注意してください。

### 出力物

- `output/都府県名.osc`: 生成された変更セットファイル。
- `都道府県名.overpass.tmp`: Overpass API で取得した対象 ID の一覧。

## 処理の仕様

### タグの変換ロジック

#### 1. addr:full (住所)
- `addr:full` = 現在の市区町村名 + v1の `KSJ2:ADS`

#### 2. operator (管理者/運営者)
v1 の `KSJ2:PubFacAdmin` タグの値に基づき変換します。
- `民間` → `operator:type=private`
- `市区町村` → `operator=都道府県名+市区町村名`
- `都道府県` → `operator=都道府県名`
- `国` / `その他` → `KSJ2:PubFacAdmin` タグをそのまま維持

### 位置変化の検証 (バリデーション)
- **変化なし ～ 30m 未満**: 通常通り処理。
- **30m 以上 ～ 50m 未満**: 処理は行いますが、`fixme` タグを付与して注意を促します。
  - 内容: `addr:fullを機械付与した時点でオリジナルと位置がxx.xm変化しています`
- **50m 以上**: データの信頼性が低いため、処理をスキップします。

## 注意事項

- **自動アップロードは行いません**: 生成された `.osc` ファイルを JOSM 等で開き、内容を十分に確認した上で手動でアップロードしてください。
- **API 負荷**: 多数のノードを処理するため、各リクエスト間に 1 秒の待機時間を設けています。大量のデータを処理する場合は時間がかかります。
- **ライセンス**: OSMデータの編集にあたっては、OSMの投稿規約およびライセンス (ODbL) を遵守してください。
